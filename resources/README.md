- `cheatsheets/04-dict-methods.ipynb` - 字典方法速查表
- `cheatsheets/05-built-in-functions.ipynb` - 內建函式速查表

- `references/best-practices.md` - 最佳實踐指南
- `references/style-guide.md` - PEP 8 風格指南

### 🛠️ Tools（工具設定）
